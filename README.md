# Final Project for II.3510 Mobile Application Development for Android | ISEP 2022/2023

## Project Description

**Project Name:** News App

**Project Description:** News App is an android app for checking news from different sources and countries using News API. You can see a preview of the news in the app and if interested check the whole article in its original webpage, you can also share this news with your friends and save them to read them later. Log in is done using your google account thanks to Firebase Database.
